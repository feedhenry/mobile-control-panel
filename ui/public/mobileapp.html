Sync SDK Integration

Getting Started

To start with an example Sync app, clone/fork the repo at https://github.com/feedhenry-templates/feedhenry-cordova-sync-app. If you already have an App, you can still follow the below instructions to integrate the Sync SDK.

SDK Config

Copy the below contents in sync-config.json in the root of the project.

{{syncconfig}}

SDK Installation

Add the Sync SDK to the App. This example is using npm for dependency management. Alternatively, you can download releases from https://github.com/feedhenry/fh-sync-js/releases. This is already done for you if starting from the example sync app.

npm install --save fh-sync-js@latest

SDK Initialisation

Load the SDK config file & initialise the Sync SDK. This example is using AngularJS. Again, this is already done for you if starting from the example sync app.

angular
  .constant('fhSync', require('fh-sync-js'))
  .constant('fhSyncConfig', require('../../sync-config.json'))

fhSync.init({
  cloudUrl: fhSyncConfig.uri
});

You can then use the Sync API in your App. For more info on how to use the Sync API, see the example app at https://github.com/feedhenry-templates/feedhenry-cordova-sync-app and the Sync API Documentation
